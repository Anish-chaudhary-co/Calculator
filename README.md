# Calculator
A simple and interactive calculator built using HTML, CSS, and JavaScript.<br>
This project performs basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division.<br>

<h1><u>Features</u></h1><br>
1.Clean and responsive UI<br>
2.Supports +, −, ×, ÷<br>
4.Error handling (division by zero, invalid input)<br>
5.Easy to understand code structure<br>

<h1><u>Technologies Used</u></h1><br>
HTML5 – Structure<br>
CSS3 – Styling<br>
JavaScript (ES6) – Logic/Functionality<br>

<h2>This project is open-source and free to use.</h2>
